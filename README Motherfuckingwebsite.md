# Mother-fucking-website
C'est la version basique du site mother fucking website sans css stylisé
index.html c'est la page d'interface qui va déboucher sur les autres pages web via un clic sur les liens correspondants
le premier lien c'est "txti" qui permet d'ouvrir(texties.html) ensuite clic sur le lien what' s txti ? qui permet d'ouvrir la page "about txti" (whatstexti.html) et le lien how to use txti? vers la page (howtomadetxties.html)
sous la meme page "txti"(texties.html) un clic vers le lien terms of service (termsofservice.html)
au niveau de la page "about txti"(whatstexti.html) deux liens sont accessibles : barry t smith qui permet d'atteindre (barry t smith.html)et adam newbold(neatnik.home.html)
le (neatnik.home.html) permet d'accéder aux liens(neatnik.work) et (neatnik.contact)
au niveau du lien how to use txti? on dirige vers la page (howtomadetxties.html)où un lien texti.es.image permet d'atteindre la page (textiesimages.html)qui contient un lien click to load images dirige vers (loadimages.html)
Pour la réalisation de ce site un ensemble d'images sont utilisés
